Introduction to Javascript - Javascript Coding Examples

For each exercise use a comment line to put the Exercise number within the script. Also place a short description of what the script is doing.

1. Create a runtime script that will process an alert( ). The alert should display your name.

2. Create a runtime script that will process a document.write( ). The script should write your name below as an H1 element.

3. Create a runtime script that will process a document.write( ). The script should write a short description of what you have learned today below this line. The content should be within a p element.

4. Create a function called displayCourse(). The function will call the alert method to display the course name. Use a runtime script to call the function just below this paragraph.

5. Use a button element to call the displayCourse()function. Add the onclick event handler to the button in order to call the function.

Notice that you can call this multiple times.

6. Create a function called writeCourseName(). The function should use the document.write() command to display the course name as an H1 element. Place an onclick event handler in this paragraph. When you CLICK THIS PARAGRAPH it will call the writeCourseName() function. Be prepared to discuss what happened.
